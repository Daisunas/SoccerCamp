[SoccerCamp](https://daisunas.github.io/SoccerCamp/)

Soccer Camp is an attendance tracker web app used to check-in kids to a soccer camp using their Order-ID number which is assigned per family. All data is self-uploaded and parsed from a json file. 

The example dataset has Order IDs 1234566-1234572. 1234567 has two family members enrolled in the camp.
